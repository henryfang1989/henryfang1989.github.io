Solution #1

Queue is FIFO and Stack is LIFO.

In order to simulate the last in and first out, we need to find a way to get the last element in the queue.

The way to get the last one is keeping pop out the element from queue and push it back to queue except for the last element in the queue.

In pop(), we do the pop out and push back loop until hit the last element, we pop out and return the element.

In top(), we do the same process as pop(), but we do not pop put the last the element. We just read the element value, then pop out and push back to queue to ensure the queue maintain the right sequence.

time complexity: push and empty:O(1), pop and top: O(n) space complexity: O(1)

Code

class MyStack(object):

    def __init__(self):
        """
        Initialize your data structure here.
        """
        from collections import deque
        self.q = deque()

    def push(self, x):
        """
        Push element x onto stack.
        :type x: int
        :rtype: None
        """
        self.q.append(x)
        

    def pop(self):
        """
        Removes the element on top of the stack and returns that element.
        :rtype: int
        """
        for _ in xrange(len(self.q)-1):
            self.q.append(self.q.popleft())
        return self.q.popleft()
        

    def top(self):
        """
        Get the top element.
        :rtype: int
        """
        for _ in xrange(len(self.q)-1):
            self.q.append(self.q.popleft())
        res = self.q[0]
        self.q.append(self.q.popleft())
        return res
        

    def empty(self):
        """
        Returns whether the stack is empty.
        :rtype: bool
        """
        return not self.q

# param_3 = obj.top()
# param_4 = obj.empty()

Solution #2

Use two queues to simulate the stack. The status of two queues are one is empty and other one contains the elements. This status will be swapped after the push()

push(): insert new element into the empty queue and pop out the elements from non-empty queue and push them to the empty queue as well. By doing that, we can see the new inserted element is always in the head of queue. Now the previous non-empty queue becomes the empty queue and ready for next push() call.

pop(): by using two queues, the pop out is quite simple, which just pop out the head element from non-empty queue.

top(): it is similar to pop()

empty(): have to check both queues

In order to tell the empty queue and non-empty queue, we use index to points to the non-empty queue. The index+1after doing push() because the status of queues get swapped.

time complexity: pushO(n) the rest of calls: O(1)` space complexity: O(1)

Code

class MyStack(object):

    def __init__(self):
        """
        Initialize your data structure here.
        """
        self.qs = [collections.deque(), collections.deque()]
        self.index = 0 # points to stack_head, the side pops out elements

    def push(self, x):
        """
        Push element x onto stack.
        :type x: int
        :rtype: None
        """
        stack_head, stack_tail = self.qs[(self.index+1)%2], self.qs[self.index%2]
        self.index += 1
        stack_head.append(x)
        while stack_tail:
            stack_head.append(stack_tail.popleft())
            

    def pop(self):
        """
        Removes the element on top of the stack and returns that element.
        :rtype: int
        """
        stack_head = self.qs[self.index%2]
        return stack_head.popleft()
        

    def top(self):
        """
        Get the top element.
        :rtype: int
        """
        stack_head = self.qs[self.index%2]
        return stack_head[0]

    def empty(self):
        """
        Returns whether the stack is empty.
        :rtype: bool
        """
        a, b = self.qs
        return not a and not b

# param_4 = obj.empty()

Solution #3

In solution #2, we use two queues. Actually, we could do it with one queue.

We know insert a new element will be placed in the end of queue. In order to make the new element to the head of queue, we can pop out all old elements from queue and push back the queue.

By doing that, the sequence of elements in the queue is same as in stack. The head of element is always the newest element.

time complexity: push:O(n) the rest of call: O(1) space complexity: O(1)

Code

class MyStack(object):

    def __init__(self):
        """
        Initialize your data structure here.
        """
        self.q = collections.deque()

    def push(self, x):
        """
        Push element x onto stack.
        :type x: int
        :rtype: None
        """
        size = len(self.q)
        self.q.append(x)
        for _ in xrange(size):
            self.q.append(self.q.popleft())

    def pop(self):
        """
        Removes the element on top of the stack and returns that element.
        :rtype: int
        """
        return self.q.popleft()

    def top(self):
        """
        Get the top element.
        :rtype: int
        """
        return self.q[0]

    def empty(self):
        """
        Returns whether the stack is empty.
        :rtype: bool
        """
        return not self.q
