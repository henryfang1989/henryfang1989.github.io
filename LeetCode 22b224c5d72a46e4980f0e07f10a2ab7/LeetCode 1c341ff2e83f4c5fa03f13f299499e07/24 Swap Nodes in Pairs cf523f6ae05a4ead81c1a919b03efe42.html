Solution #1

Iterative approach: Swap two nodes at each time and reconnect them back to original linked list

  • Create a swap function to take four adjacent nodes at each time
  • Swap two nodes in the middle and reconnect those four nodes
  • Move to next fours adjacent nodes to do the same process.

time complexity: O(N) space complexity: O(1)

Code

# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def swapPairs(self, head):
        """
        :type head: ListNode
        :rtype: ListNode
        """
        # swap node1 and node2
        # reconnect the swapped linked list with original linked list
        def swap(node0, node1, node2, node3):
            node0.next = node2
            node2.next = node1
            node1.next = node3
            
        prev = dummy = ListNode(0)
        dummy.next = head
        while head and head.next:
            next_head = head.next.next
            swap(prev, head, head.next, head.next.next)
            prev = head
            head = next_head
        return dummy.next

Solution #2

Recursive approach. Letâ€™s assume swapPairs return a new linked list after swap.

  • Take first and second as two nodes that are going to be swapped.
  • We make a recursive call to get a new linked list for nodes after second
  • We swap the first and second and connect new linked list with first and return second

time complexity: O(n) space complexity: O(n)

Code

# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def swapPairs(self, head):
        """
        :type head: ListNode
        :rtype: ListNode
        """
        if not head or not head.next:
            return head
        
        first = head
        second = head.next
        
        first.next = self.swapPairs(second.next)
        second.next = first
        
        return second
