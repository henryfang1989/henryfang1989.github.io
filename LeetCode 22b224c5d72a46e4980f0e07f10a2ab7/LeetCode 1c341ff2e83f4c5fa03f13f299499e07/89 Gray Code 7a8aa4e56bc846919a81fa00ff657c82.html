Solution #1

In order to make the adjacent binary numbers differ in only 1 bit, we have to carefully construct the binary number bit by bit.

The way to construct the binary number is appending 0 and 1 bit to the same existing binary number. Because we share the same binary number, it is guaranteed that only the new appending bits are different.

Next step is appending 0 and 1 bit to the next existing binary number. In order to keep the adjacent binary number differ in only 1 bit. The order of appending the new bit is 1 and 0, so that we keep all new binary numbers in 1 bit difference.

Example:

Existing binary numbers is [x, y]

For x, we append 0 and 1 → x0, x1 → those two binary numbers differ only in 1 bit

For y, we append 1 and 0 → y1, y0 → those y1 and y0 are 1 bit difference. We also guarantee x1 and y1 are 1 bit different because x and y are 1 bit deferent.

time complexity: O(2^n) space complexity: O(2^n)

Code

class Solution(object):
    def grayCode(self, n):
        """
        :type n: int
        :rtype: List[int]
        """
        res = [0]
        for i in xrange(n):
            temp = []
            for j, num in enumerate(res):
                if j % 2 == 0:
                    temp.append(2*num)
                    temp.append(2*num+1)
                else:
                    temp.append(2*num+1)
                    temp.append(2*num)
            res = temp
        return res

Solution #1

Instead of appending the new bit, the other way to construct it reverse the list of existing binary number and prepend the 0 and 1. Keep doing it until n times.

time complexity: O(2^n) space complexity: O(2^n)

Code

class Solution(object):
    def grayCode(self, n):
        """
        :type n: int
        :rtype: List[int]
        """
        res = [0]
        for i in xrange(n):
            sz = len(res)
            for j in xrange(sz-1, -1, -1):
                res.append(res[j] | 1 << i)
        return res
