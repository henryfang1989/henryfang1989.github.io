Solution #1

The approach is very simple. Just access the numbers in reader one by one. Increase index by 1 and access the next number until we find the target.

Note: we have to check if get() returns 2147483647 as run out of array.

time complexity: O(n) space complexity: O(1)

Code

# """
# This is ArrayReader's API interface.
# You should not implement it, or speculate about its implementation
# """
#class ArrayReader(object):
#    def get(self, index):
#        """
#        :type index: int
#        :rtype int
#        """

class Solution(object):
    def search(self, reader, target):
        """
        :type reader: ArrayReader
        :type target: int
        :rtype: int
        """
        index = 0
        limit = 2147483647
        while reader.get(index) != target and reader.get(index) != limit:
            index += 1
        return index if reader.get(index) != limit else -1

Solution #2

Use binary search to find the index. The important info given in question is number of integers in array are less than 10000. It lets us know the boundary of binary search.

the get() returns a value which has 4 cases.

  1. value < target
  1. value > target
  1. value == target
  1. value == 2147483647 â†’ out of index

Actually we could combine case 2 and case 4 because both of values are > target and we should reduce the mid in binary search.

Therefore, we could use standard binary search to find the index.

Note: we have to check s and e positions after while loop because those two positions may not be checked in while loop.

time complexity: O(logn) space complexity: O(1)

Code

# """
# This is ArrayReader's API interface.
# You should not implement it, or speculate about its implementation
# """
#class ArrayReader(object):
#    def get(self, index):
#        """
#        :type index: int
#        :rtype int
#        """

class Solution(object):
    def search(self, reader, target):
        """
        :type reader: ArrayReader
        :type target: int
        :rtype: int
        """
        s, e = 0, 10000
        limit = 2147483647
        while s + 1 < e:
            mid = (s + e) / 2
            if reader.get(mid) > target or reader.get(mid) == limit:
                e = mid
            elif reader.get(mid) < target:
                s = mid
            else:
                return mid
  
        # s or e positions may not be checked
        if reader.get(s) == target:
            return s
        if reader.get(e) == target:
            return e
        return -1
