Solution #1

This question is a linked list manipulation. The question tells us the values in linked list are sorted. Therefore, we can check the adjacent nodes to determine if two nodes are duplicated or not.

We create a dummy node to hold the head of result. Use t to connect the unique node every time.

Use head to iterate each node in linked list.

We always add the first node in to result and skip all duplicated nodes after the first node.

time complexity: O(n) space complexity: O(1)

Code

# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def deleteDuplicates(self, head):
        """
        :type head: ListNode
        :rtype: ListNode
        """
        t = dummy = ListNode(-1)
        
        while head:
            # connect the unique node to the result
            t.next = head
            t = head
            head = head.next
            t.next = None
            # skip all duplicated nodes
            while head and head.val == t.val:
                head = head.next
        return dummy.next

Solution #2

This approach is using recursion.

In recursive call, we skip all duplicated nodes, whose value is same as current node. Then make a call again to process the next node.

But the stack maybe overflow is the linked list is too long

time complexity: O(n) space complexity: O(n)

Code

# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def deleteDuplicates(self, head):
        """
        :type head: ListNode
        :rtype: ListNode
        """
        if not head or not head.next:
            return head
        # skip all duplicated nodes
        while head.val == head.next.val:
            head.next = head.next.next
            if not head.next:
                break
        self.deleteDuplicates(head.next)
        return head
