Solution #1

This approach is create a function to check if a number is prime number. Then we iterate the numbers from 2 → n to find all prime numbers.

The way to check if a number is prime number is find if the number is dividable by any number between [2, sqrt(num)]

time complexity: O(n*sqrt(n)) space complexity: O(1)

Code

class Solution(object):
    def countPrimes(self, n):
        """
        :type n: int
        :rtype: int
        """
        import math
        def is_prime(num):
            for i in xrange(2, int(math.sqrt(num))+1):
                if num % i == 0:
                    return False
            return True
        
        return sum(is_prime(i) for i in xrange(2, n))

Solution #2

The solution 1, the time complexity is too high and causes the TLE.

Instead of checking every number, we could find out those non-prime number first and left numbers are prime numbers.

We use a length of n array to indicate each index of array is prime or not prime.

The way to find out those non-prime numbers is using all possible numbers * a prime number as long as the result is < n . When we find the prime number, we only have to search the number ≤ sqrt(n) because the number after sqrt(n) is symmetric with numbers before it.

time complexity: O(n) space complexity: O(n)

Code

class Solution(object):
    def countPrimes(self, n):
        """
        :type n: int
        :rtype: int
        """
        if n < 2:
            return 0
        is_prime = [1] * n
        is_prime[0] = is_prime[1] = 0
        for i in xrange(2, n):
            if is_prime[i] == 1:
                index = 2
                while index * i < n:
                    is_prime[index*i] = 0
                    index += 1
        return sum(is_prime)

Solution #3

In this approach, it does some optimizations.

  1. we don't have to search the prime numbers all the way to n because the results of non-prime numbers are symmetric by search them using primes before sqrt(n) and after sqrt(n)
  1. when we search the non-prime numbers, we don't have to search from prime * 2 because we always do the same process for 2, we could just search it from prime * prime.

time complexity: O(n) space complexity: O(n)

Code

class Solution(object):
    def countPrimes(self, n):
        """
        :type n: int
        :rtype: int
        """
        import math
        if n < 2:
            return 0
        is_prime = [1] * n
        is_prime[0] = is_prime[1] = 0
        limit = int(math.sqrt(n))+1
        for i in xrange(2, limit):
            if is_prime[i] == 1:
                index = 2
                while index * i < n:
                    is_prime[index*i] = 0
                    index += 1
        return sum(is_prime)
