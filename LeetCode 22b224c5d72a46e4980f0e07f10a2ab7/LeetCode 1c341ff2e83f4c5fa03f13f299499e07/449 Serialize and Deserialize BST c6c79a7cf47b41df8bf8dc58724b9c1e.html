Solution #1

This approach uses in-order traverse to visit each node in the tree.

Serialize():

  • In-order traverse through the tree and add the value as string into result
  • If hit a node is None, add # as a sign of None node
  • Use , as delimiter to separate the values from different nodes

Deserialize():

  • Use in-order traverse through the tree and pop out the head of queue and use it as value for current node.
  • If hit a value is #, we know it is None node, just return None
  • Convert a string into a deque so it is easy to access the value by order.

time complexity: serializeO(n), deserialize O(n) space complexity: serializeO(logn), deserialize O(n)

Code

# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Codec:

    def serialize(self, root):
        """Encodes a tree to a single string.
        
        :type root: TreeNode
        :rtype: str
        """
        if not root:
            return '#'
        return str(root.val) + ' ' + self.serialize(root.left) + ' ' + self.serialize(root.right)

    def deserialize(self, data):
        """Decodes your encoded data to tree.
        
        :type data: str
        :rtype: TreeNode
        """
        data = collections.deque(data.split())
        def helper(data):
            val = data.popleft()
            if val == '#':
                return None
            root = TreeNode(int(val))
            root.left = helper(data)
            root.right = helper(data)
            return root
        return helper(data)

Solution #2

post-order traverse + BST check

  • Do post-order traverse and add the nodeâ€™s value to result.
  • No need add special character for None child node because we can use the value range to determine if next value is its child node
  • In decode(), we could just pop out the value from list and use that value as root node value because we do the post-order traverse.

time complexity: O(N) space complexity: O(N)

Code

# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Codec:

    def serialize(self, root):
        """Encodes a tree to a single string.
        
        :type root: TreeNode
        :rtype: str
        """
        if not root:
            return ""
        # the reason for ding post-order traverse
        # is for list to do the pop()
        return self.serialize(root.left) + " " + self.serialize(root.right) + " " + str(root.val)

    def deserialize(self, data):
        """Decodes your encoded data to tree.
        
        :type data: str
        :rtype: TreeNode
        """
        data = data.split()
        def helper(data, s, e):
            if not data:
                return None
            val = int(data[-1])
            # because it is a BST, so the
            # value range could tell if the node
            # is a child node of current node.
            # No need to set None node as #
            if not s < val < e:
                return None
            # it is post-order traverse,
            # so the pop out node is the root node value
            val = int(data.pop())
            root = TreeNode(val)
            root.right = helper(data, val, e)
            root.left = helper(data, s, val)
            return root
        return helper(data, float('-inf'), float('inf'))
