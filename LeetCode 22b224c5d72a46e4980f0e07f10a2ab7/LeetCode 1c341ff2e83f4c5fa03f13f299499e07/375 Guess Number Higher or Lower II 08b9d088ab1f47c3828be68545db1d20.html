Solution #1

This question asks us to find a way to guess the number that uses the minimum amount of money to guarantee a win.

In order to guarantee a win, we have to always choose the worst case when we don't know where the guess number is. For instance, when we choose a number i, we have two options for the next number, which are the left side and right side of i. In order to cover the worst case, we have to choose the side that has a larger cost.

However, we could choose the i intelligently. We could choose the i that makes the total cost minimized.

We use the dp two dimensions array to save the results to prevent duplicated calculations.

time complexity: O(n3)O(n^3)O(n3)﻿ space complexity: O(n2)O(n^2)O(n2)﻿

Code

class Solution(object):
    def getMoneyAmount(self, n):
        """
        :type n: int
        :rtype: int
        """
        def helper(s, e, dp):
            if dp[s][e] != -1:
                return dp[s][e]
            if s >= e:
                return 0
            res = float('inf')
            for i in xrange((s+e)/2, e):
                # after choosing a number, we always conside the worst case for rest of numbers
                # but we still consider the min cost by choosing different number
                res = min(res, i + max(helper(s, i-1, dp), helper(i+1, e, dp)))
            dp[s][e] = res
            return res
        
        dp = [[-1 for i in xrange(n+1)] for j in xrange(n+1)]
        return helper(1, n, dp)

Solution #2

This approach uses bottom to up DP solution.

The dp formula is dp[i][j]=min(k+max(dp[i][k−1],dp[k+1][j]))dp[i][j] = min(k + max(dp[i][k-1], dp[k+1][j]))dp[i][j]=min(k+max(dp[i][k−1],dp[k+1][j]))﻿ The dp[i][j]dp[i][j]dp[i][j]﻿ represents the minimum cost for guarantee a win in range [i, j].

Be careful about the dp length. kkk﻿ should be 1~n, and we have to make sure k−1k-1k−1﻿ and k+1k+1k+1﻿ in a valid range. Therefore, the dp length should be n+2n+2n+2﻿

time complexity: O(n3)O(n^3)O(n3)﻿ space complexity: O(n2)O(n^2)O(n2)﻿

Code

class Solution(object):
    def getMoneyAmount(self, n):
        """
        :type n: int
        :rtype: int
        """
        
        dp = [[float('inf') for i in xrange(n+2)] for j in xrange(n+2)]
        
        for i in xrange(n+1):
            for j in xrange(n+1):
                if i >= j:
                    dp[i][j] = 0
            
        for i in xrange(n, 0, -1):
            for j in xrange(i+1, n+1):
                for k in xrange((j-i)/2, j+1):
                    # print i, j, k
                    dp[i][j] = min(dp[i][j], k + max(dp[i][k-1], dp[k+1][j]))
        return dp[1][n]
