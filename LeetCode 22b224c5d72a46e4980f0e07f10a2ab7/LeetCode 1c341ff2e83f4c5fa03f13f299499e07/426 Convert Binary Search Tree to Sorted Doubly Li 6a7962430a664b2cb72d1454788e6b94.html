Solution #1

Use divide and conquer to convert a BST to a circular sorted double linked list.

We build a helper function to convert a BST to a sorted double linked list.

helper function returns the head and tail of linked list after conversion.

Once we get the results from both subtrees, we connect two linked lists with root node.

Note: we have to be careful about the empty linked list returned from subtree

time complexity: O(n) space complexity: O(logn)

Code

"""
# Definition for a Node.
class Node(object):
    def __init__(self, val, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
"""

class Solution(object):
    def treeToDoublyList(self, root):
        """
        :type root: Node
        :rtype: Node
        """
        # divide and conquer
        def helper(root):
            if not root:
                return root, root
            
            # get convert results from left and right subtrees
            la, lb = helper(root.left)
            ra, rb = helper(root.right)
            
            # connect root with left part
            root.left = lb
            if la:
                lb.right = root
                la.left = None
                s = la
            else:
                s = root
            # connect root with right part
            root.right = ra
            if ra:
                ra.left = root
                rb.right = None
                e = rb
            else:
                e = root
            # return new start and end of double linked list
            return s, e
          
        s, e = helper(root)
        # convert to circular double linked list
        if s and e:
            s.left = e
            e.right = s
        return s

Solution #2

This approach is using in-order traverse tree to access each node.

We have global variable first to track the linked list's head.

The global variable last tracks previous processing node.

We connect the current node with last node when visit each node.

Then we set the current node as last node for next node process.

The in-order traverse visits the BST from smallest node to largest node, so all nodes are connected in sorted order.

Lastly, we have to connect the first to last to construct the circular double linked list.

time complexity: O(n) space complexity: O(logn)

Code

"""
# Definition for a Node.
class Node(object):
    def __init__(self, val, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
"""

class Solution(object):
    def __init__(self):
        self.first = self.last = None
    def treeToDoublyList(self, root):
        """
        :type root: Node
        :rtype: Node
        """
        def inorder_dfs(root):
            if not root:
                return
            inorder_dfs(root.left)
            if self.last:
                self.last.right = root
                root.left = self.last
            else:
                self.first = root
            self.last = root
            inorder_dfs(root.right)
        
        inorder_dfs(root)
        if self.first and self.last:
            self.first.left = self.last
            self.last.right = self.first
        return self.first

Solution #3

This approach is similar to solution 2. But it is use iterative way to traverse the tree.

time complexity: O(n) space complexity: O(logn)

Code

"""
# Definition for a Node.
class Node(object):
    def __init__(self, val, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
"""

class Solution(object):
    def treeToDoublyList(self, root):
        """
        :type root: Node
        :rtype: Node
        """
        stack = []
        first = last = None
        while True:
            while root:
                stack.append(root)
                root = root.left
            if not stack:
                if first and last:
                    first.left = last
                    last.right = first
                return first
            node = stack.pop()
            if last:
                last.right = node
                node.left = last
            else:
                first = node
            last = node
            root = node.right
