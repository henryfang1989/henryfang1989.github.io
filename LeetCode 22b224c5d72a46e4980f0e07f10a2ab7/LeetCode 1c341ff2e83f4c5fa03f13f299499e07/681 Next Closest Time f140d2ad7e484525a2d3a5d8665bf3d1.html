Solution #1

Simulate the real clock by keeping add 1 min to given time.

Check the digits in new time. If all digits in new time exists in given time, return the time

time complexity: O(1) space complexity: O(1)

Code

class Solution(object):
    def nextClosestTime(self, time):
        """
        :type time: str
        :rtype: str
        """
        h, m = time.split(":")
        total = int(h) * 60 + int(m)
        # convert total mins to a time string
        def convert(mins):
            h, m = mins/60, mins%60
            h = ('0' if h < 10 else "") + str(h)
            m = ('0' if m < 10 else "") + str(m)
            return h+":"+m
        # keep adding 1 min to total to simulate the real clock
        while True:
            total += 1
            total %= 24*60
            digits = convert(total)
            if all([x in time for x in digits]):
                return digits

Solution #2

Use the digits from time and construct all valid and possible digits combinations. Check the combinations and find closest one.

time complexity: O(1) space complexity: O(1)

Code

class Solution(object):
    def nextClosestTime(self, time):
        """
        :type time: str
        :rtype: str
        """
        digits = [x for x in time if x.isdigit()]
        start = int(digits[0] + digits[1]) * 60 + int(digits[2] + digits[3])
        
        diff = float('inf')
        res = time
        # construct all possible times
        for a in digits:
            for b in digits:
                for c in digits:
                    for d in digits:
                        if int(a+b) > 23 or int(c+d) >= 60:
                            continue
                        cur = int(a+b) * 60 + int(c+d)
                        t = (cur - start) % (24*60)
                        # find closest one, but not itself
                        if t < diff and t != 0:
                            diff = t
                            res = a + b + ":" + c + d
        return res
