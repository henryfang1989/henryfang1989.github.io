Solution #1

Use a hashmap to count the numbers

return the unique number with counts > 1

time complexity: O(n) space complexity: O(n)

Code

class Solution(object):
    def findDuplicates(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        d = collections.Counter(nums)
        return [x for x, y in d.items() if y > 1]

Solution #2

This approach is using the sign of numbers in array to indicate if that number has been visited.

We know the numbers in array in [1, n] range. We could use them as index of current array.

We iterate the numbers and use the number as index to visit that position. If we first time visit that position, we modify the sign of number to make the number become a negative value.

We use negative to represent the position has been visited before, which mean the number used as index is duplicated number.

Note: we have to get abs(number)-1 value as index, because the number may be modified by other numbers already.

Because we reuse the array, so there is no extra space

time complexity: O(n) space complexity: O(1)

Code

class Solution(object):
    def findDuplicates(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        res = []
        for num in nums:
						# have to minus 1 to convert to index
            index = abs(num)-1
						# if this position has been visited
						# we find the duplicated number
            if nums[index] < 0:
                res.append(index+1)
						# if it is the first time
						# modify the value pointed by current number
            else:
                nums[index] = -nums[index]
        return res
