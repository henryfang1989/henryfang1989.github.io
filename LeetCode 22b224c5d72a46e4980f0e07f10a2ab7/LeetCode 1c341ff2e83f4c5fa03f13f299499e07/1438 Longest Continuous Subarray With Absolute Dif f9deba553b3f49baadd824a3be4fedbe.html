Solution #1

Use sliding window + monotonic deque.

We use sliding window to extend the right side of window. Then we check if the condition is met for new element. If not, shrink the window by moving the left side of window.

Two monotonic deques are used to get the max and min values by O(1). We could check if new element is qualified by checking the abs diff between max and min values.

time complexity: O(n) space complexity: O(n)

Code

class Solution(object):
    def longestSubarray(self, nums, limit):
        """
        :type nums: List[int]
        :type limit: int
        :rtype: int
        """
        # two montonic queues to get max/min value by O(1)
        min_queue = collections.deque()
        max_queue = collections.deque()
        start = 0
        res = 0
        # sliding window to find longest window
        for i, num in enumerate(nums):
            while min_queue and num < min_queue[-1]:
                min_queue.pop()
            while max_queue and num > max_queue[-1]:
                max_queue.pop()
            
            min_queue.append(num)
            max_queue.append(num)
            
            while (max_queue[0] - min_queue[0]) > limit:
                if max_queue[0] == nums[start]: max_queue.popleft()
                if min_queue[0] == nums[start]: min_queue.popleft()
                start += 1
            
            res = max(res, i-start+1)
        return res

Solution #2

Sliding window + two heaps

In this approach, we use two heaps to maintains the max and min values.

Use the same idea in solution 1. We add the new element into two heaps and compare the abs diff between max and min values with limit.

time complexity: O(nlogn) space complexity: O(n)

Code

class Solution(object):
    def longestSubarray(self, nums, limit):
        """
        :type nums: List[int]
        :type limit: int
        :rtype: int
        """
        import heapq
        # use two heaps to maintain the max and min value
        min_h = []
        max_h = []
        
        k = res = 0
        for i, num in enumerate(nums):
            heapq.heappush(min_h, [num, i])
            heapq.heappush(max_h, [-num, i])
            # check if the condition is met when a new element is added
            while -max_h[0][0] - min_h[0][0] > limit:
                k = min(min_h[0][1], max_h[0][1]) + 1
                while min_h[0][1] < k: heapq.heappop(min_h)
                while max_h[0][1] < k: heapq.heappop(max_h)
            
            res = max(res, i - k + 1)
        return res
